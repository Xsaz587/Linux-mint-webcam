Сетевой Видеопоток с Honor

Используйте телефон как IP-камеру. Просмотр ведется прямо на Linux Mint.

Инструкции по настройке Honor:

  1. Установите приложение **"IP Webcam"** (или аналогичное) из Google Play на Honor.
  2. Запустите приложение и нажмите **"Start Server"** (Запустить сервер).
  3. **Для подключения через Wi-Fi:** Введите IP-адрес, который покажет приложение, и порт (например, `192.168.1.10` и `8080`).
  4. **Для подключения через USB (рекомендуется):**
    • Включите режим USB-отладки на телефоне.
    • Подключите телефон к Linux Mint по USB.
    • В терминале выполните команду: `adb forward tcp:8080 tcp:8080`
    • В поля ниже введите: **IP-адрес: `127.0.0.1` (localhost)** и **Порт: `8080`**.
Введите IP-адрес и нажмите "Начать просмотр".
Для работы приложения требуется, чтобы на телефоне Honor был запущен сервер IP-веб-камеры.
